# Images pour les jeux

## Structure des dossiers

- `sharks/` - Images des requins pour le jeu "Rythme Requin"
- `fish.png` - Poisson normal (distracteur) pour le jeu "Rythme Requin"

## Image de poisson requis

Veuillez ajouter :
- `fish.png` - Un poisson coloré simple (pas un requin) pour servir de distracteur dans le jeu

**Spécifications** :
- Format : PNG avec transparence
- Taille : Environ 150-300px de largeur
- Orientation : Poisson orienté vers la droite (→)
