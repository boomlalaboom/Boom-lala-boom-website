# Images pour les jeux

## Structure des dossiers

- `sharks/` - Images des requins pour le jeu "Rythme Requin"
- `fish.png` - Poisson normal (distracteur) pour le jeu "Rythme Requin"
- `net.svg` - ✅ Filet de pêche (créé)
- `ocean-background.png` - Fond marin pour le jeu "Rythme Requin"

## Images requises

### Pour le dossier `sharks/` :
- `baby-shark.png` (ou .svg) - Bébé requin
- `papa-shark.png` (ou .svg) - Papa requin
- `mama-shark.png` (ou .svg) - Maman requin
- `grandma-shark.png` (ou .svg) - Grand-mère requin
- `grandpa-shark.png` (ou .svg) - Grand-père requin

### Pour le dossier `images/` :
- `fish.png` - Un poisson coloré simple (pas un requin) pour servir de distracteur
- `ocean-background.png` - Le fond marin

**Spécifications** :
- Format : PNG avec transparence (ou SVG)
- Taille : Environ 150-300px de largeur pour requins/poissons
- Orientation : Tous orientés vers la droite (→)
