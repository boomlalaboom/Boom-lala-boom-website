## Instructions

1. Va sur `http://localhost:5173/admin` (ou ton URL de production)
2. Connecte-toi avec tes identifiants admin
3. Clique sur l'onglet **"Characters"**
4. Pour chaque personnage ci-dessus, clique sur **"Add New"**
5. Remplis tous les champs avec les valeurs indiquées
6. Clique sur **"Save"**
7. Répète pour tous les 11 personnages

**Note:** Les images sont déjà dans le dossier `/public/animals/` et `/public/images/sharks/`, donc tu peux utiliser directement les chemins indiqués.
